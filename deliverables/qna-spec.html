E-05저장 검증[저장] 클릭 시 입력 누락제목 미입력 → "제목을 입력해주세요." / 파일 미선택 → "파일을 선택해주세요." 토스트 후 중단

8비고
본 문서의 대상 화면(qna.html)은 UI 시안(프로토타입)입니다. 구현 시 자료 목록 조회·다운로드는 파일 저장소 및 다운로드 API 연동으로, 자료 추가·수정은 파일 업로드 API + 서버 측 관리자 권한 검증으로 대체합니다(시안의 관리자 토글은 화면 확인용 임시 장치). 업로드 시 용량·등록일 자동 처리, 확장자 기반 유형 추정 동작은 시안과 동일하게 유지합니다.
자료 삭제 기능은 정책상 제공하지 않습니다(게시 자료의 이력 보존 — 수정·교체만 허용). 헤더·사이드바·토스트는 전 화면 공통 영역으로, 별도 공통 컴포넌트로 관리합니다(상세는 공통 정의 참조).
