7.1 · 페이지 구성 (change.html 고유)

페이지 고유 스타일은 <style> 블록에 정의 — 시연용 버튼(.demo-btn), 밑줄형 메인 탭(.maintab/.mtab), 탭 패널(.tab-panel), 대상 VDI 라디오 리스트(.vdi-select/.vdi-opt 계열), 증설 비교 그리드(.sc-table/.sc-row 계열), 현재 사용기간 텍스트(.info-text).

구성 요소역할
VDI_ALL / VDI_LIST보유 가상PC 3대 mock(5장). VDI_LIST는 시연 토글에 따라 전체 또는 기본 지급만으로 구성
CPU_OPTS · MEM_OPTS · DISK_OPTS증설 자원 옵션 정의 (6/8 Core · 8/16 GB · 150/200 GB)
state화면 상태 단일 객체 — 현재 탭(tab) · 선택 VDI 인덱스(vdi) · 증설 목표값(e_cpu/e_mem/e_disk)
extendable() / blockReason()신청 가능 판정 단일 출처 — 잠김·기본 지급 차단 및 차단 사유 문구 반환(6장 매트릭스 구현)
STEPS / renderStepper()탭별 처리 단계 정의(연장 3단계·증설 4단계) 및 stepper 렌더(1단계 active)
renderVdiCards()대상 VDI 라디오 리스트 렌더 + 선택/차단 클릭 처리
syncForms()선택 VDI 기준 폼 동기화 — 연장(현재 기간·종료일 min/기본값·PILL 초기화, 연장 가능 VDI 없으면 빈 화면 전환) + 증설 목표값 리셋
specRow() / renderExpandGrid()증설 비교 그리드(현재 → 목표 콤보) 렌더 — 하향 옵션 disabled · "(현재)" 표기 · "증설" 배지
submitRequest() / finish()탭별 검증(E-07/E-08) → 접수 토스트 → 1.4초 뒤 approval.html 이동
toggleDemoMode() / selectExtendable()시연용 보유 상태 전환 및 연장 탭 선택 보정(데모 전용 — 구현 제외)
날짜 헬퍼 fmtISO/fmtDot/addMonths/nextDayISO↔점 표기 변환 · 개월 가산 · 익일 계산(종료일 min 산정)
7.2 · 공통 의존 (common.js · common.css)
출처사용 항목
common.js (동작)헤더/사이드바 주입(data-header · data-sidebar="change" → renderHeader/renderSidebar) · showToast()(검증/차단/접수 안내) · goHome()/logout()(헤더) · 사용자/헬프데스크/메뉴 설정(PORTAL_USER · SERVICE_DESK · NAV_SECTIONS)
common.css (스타일)레이아웃(.top-header/.layout/.sidebar/.main) · .breadcrumb · .page-header · .reveal · 폼 카드(.fc 계열) · 폼 행(.frow 계열·.field-inline) · 입력(.fi/.fs/.fta/.req) · 기간 PILL(.period-pills/.qp) · 처리 단계(.stepper/.step/.step-no/.step-lbl/.step-bar) · 액션 바(.action-bar/.action-hint/.action-btns) · 버튼(.btn/.btn-primary/.btn-outline) · .warn-box · .empty-state · 상태 점(.sdot 계열) · .toast
8연관 화면
  • 메인 포탈 (portal.html) — breadcrumb [홈] · [취소] 클릭 시 이동. 보유 VDI 3대 데이터 정합 유지
  • 결재 현황 (approval.html) — 제출 성공 시 1.4초 뒤 자동 이동. 본 화면 신청 건은 구분 연장 / 증설로 목록에 표시
  • 결재 상세 (approval-detail.html) — 연장/증설 구분별 결재 플로우와 본 화면 stepper 단계(연장 3단계·증설 4단계) 정합 유지
9비고
본 문서의 대상 화면(change.html)은 UI 시안(프로토타입)입니다. 구현 시 연동 대상 — 보유 VDI 조회, 사용 연장 신청 API, 자원 증설 신청 API, 자원 옵션(증설 가능 사양) 정책 조회. 시안의 mock 데이터·접수 토스트·페이지 이동 타이머는 실제 로직으로 대체합니다.
페이지 헤더 우측 시연용 버튼(toggleDemoMode · ?demo=noext)은 데모 전용으로, 실제 구현 범위에서 제외합니다. 헤더·사이드바·토스트는 전 화면 공통 영역으로 별도 공통 컴포넌트로 관리합니다(상세는 공통 정의 참조).
