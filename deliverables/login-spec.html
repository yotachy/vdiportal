9비고
본 문서의 대상 화면(login.html)은 UI 시안(프로토타입)으로, 화면 구성·동선 확인용입니다. 실제 인증·세션·OTP 발송·계정 잠금 등은 구현 시 연동 대상이며, 시안의 임시 동작(안내 메시지, 인증 플래그 등)은 실제 로직으로 대체합니다. 시안에서는 시연을 위해 로그인 성공 시 항상 접속 프로그램 설치 확인(install.html)을 경유하며, 실서비스에서는 설치된 경우 메인 포탈로 직행합니다.
