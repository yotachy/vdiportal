8비고
본 문서의 대상 화면(login.html)은 UI 시안(프로토타입)으로, 화면 구성·동선 확인용입니다. 실제 인증·세션·OTP 발송 등은 수행사 개발 시 연동 대상이며, 시안의 임시 동작(안내 메시지, 인증 플래그 등)은 실제 로직으로 대체합니다.
2차 인증 수단(앱알리미 OTP), OTP 유효 시간(약 3분), 로그인 실패 잠금 정책(5회) 등 세부 보안 정책 값은 협의 후 확정합니다.
