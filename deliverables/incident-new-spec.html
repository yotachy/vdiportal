9비고
본 문서의 대상 화면(incident-new.html)은 UI 시안(프로토타입)입니다. 구현 시 [신고 접수]는 장애 접수 API 연동으로 대체하고(접수번호 채번·신고자 자동 기록 포함), 첨부 영역은 실제 파일 업로드로 대체하되 업로드 정책(허용 확장자 PNG·JPG·PDF·LOG·TXT, 건당 최대 10MB, 첨부 건수 제한)을 서버에서 재검증해야 합니다. 시안의 예시 파일 첨부·toast 안내는 임시 동작입니다.
