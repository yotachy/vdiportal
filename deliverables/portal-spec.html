6상태 · 분류 정의 (VDI 상태 4종)

가상PC 상태는 전원 ON · 전원 OFF · 점검 중 · 미사용 잠김 4종으로 고정됩니다. 아래 표는 소스 구현 시 참고용 매핑입니다(시안 portal.html 기준).

상태state 키CSS 클래스색상 토큰점멸(pulse)설명 · 가능 동작
전원 ONon.vw-status.on녹색 var(--success)점멸 O정상 운영. 접속·일반/강제 재기동·장애신고 가능
전원 OFFoff.vw-status.off회색 var(--kb-gray)점멸 X전원 꺼짐. 접속 전 기동 필요
점검 중maint.vw-status.maint황색 var(--warning) · 텍스트 var(--ink)점멸 X점검/유지보수 중. 접속 제한될 수 있음
미사용 잠김locked.vw-status.locked적색 var(--danger)점멸 X장기 미사용(90일 이상 미접속) 자동 잠금. 접속 차단(모니터 잠금 아이콘 + "접속 제한") · 일반/강제 재기동 숨김 · 잠금 안내(warn-box) 표시 · 잠금해제 요청(부서장 승인)·장애신고만 가능
각 가상PC 데이터의 state 값(on/off/maint/locked)에 따라 renderVdi()가 .vw-status 요소의 클래스와 라벨 텍스트(전원 ON / 전원 OFF / 점검 중 / 미사용 잠김)를 토글하고, locked일 때는 접속 차단·재기동 버튼 숨김·잠금 안내·잠금해제 요청 버튼 노출을 함께 처리합니다. 실제 구현에서는 이 state 값을 가상PC 전원/운영 상태 API 응답으로 대체하면 됩니다. 점멸은 on에서만 적용(off·maint·locked는 정지).
