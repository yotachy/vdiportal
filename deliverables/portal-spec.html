6화면 상태 (VDI 상태)

가상접속 모니터의 상태는 전원 ON · 점검 중 · 전원 OFF 3종으로 고정됩니다. 아래 표는 수행사 소스 구현 시 참고용 매핑입니다(시안 portal.html 기준).

상태state 키CSS 클래스색상 토큰점멸(pulse)설명 · 가능 동작
전원 ONon.vw-status.on녹색 var(--success)점멸 O정상 운영. 접속·재기동·장애신고 가능
점검 중maint.vw-status.maint황색 var(--warning) · 텍스트 var(--ink)점멸 X점검/유지보수 중. 접속 제한될 수 있음
전원 OFFoff.vw-status.off회색 #5C564E점멸 X전원 꺼짐. 접속 전 기동 필요
각 가상PC 데이터의 state 값(on/maint/off)에 따라 renderVdi()가 .vw-status 요소의 클래스(on/maint/off)와 라벨 텍스트(전원 ON / 점검 중 / 전원 OFF)를 토글합니다. 실제 구현에서는 이 state 값을 가상PC 전원/운영 상태 API 응답으로 대체하면 됩니다. 점멸은 on에서만 적용(maint·off는 정지).

8비고
본 문서의 대상 화면(portal.html)은 UI 시안(프로토타입)입니다. 가상PC 접속·재기동·자원 정보·외부 시스템 연계 등은 수행사 개발 시 연동 대상이며, 시안의 임시 동작(안내 메시지 등)은 실제 로직으로 대체합니다.
