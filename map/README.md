# 스쿱포지 & 스쿱보드 — MoneyScoop 분석 도구

`map/` 폴더는 **MoneyScoop 브랜드의 웹 기반 분석 도구 두 종**을 담습니다. 순수 HTML·CSS·바닐라 JS로 구현되어 빌드 도구 없이 브라우저에서 바로 동작합니다.

> KB손해보험 VDI 포탈과는 **무관한 별도 제품**입니다(과거 도구에서 리브랜드).

| 도구 | 파일 | 설명 |
|---|---|---|
| **스쿱포지 (Scoop Forge)** ★ | `forge.html` + `forge-core.js` | 노드 전략 보드 + 라이브 차트를 합친 기술적 분석 도구. 지표 30종을 조립해 예측·시그널·국면을 산출. **현재 주력 제품** |
| **스쿱보드 (Scoop Board)** | `map.html` | 자유 캔버스 노드 다이어그램 빌더(GitMind 스타일) |

---

## 실행

- **바로 열기**: `forge.html` 또는 `map.html`을 브라우저로 직접 열면 동작(메모리 모드 — 새로고침 시 초기화).
- **서버 기능**(다중 저장·이미지 분리저장·티커 실데이터): PHP 호스팅 필요(`forge-api.php`·`api.php`). 배포 환경에선 자동 저장·불러오기.
- 외부 의존성: Pretendard 폰트(CDN) 한 개뿐. 그 외 라이브러리 없음.

---

## 스쿱포지 (Scoop Forge)

가로 **4패널** 구성:

1. **종목** — 워치리스트(좌측 사이드바)
2. **티커** — 중앙 보드: 타임프레임 매트릭스 · 예측 시점별 · 오실레이터 서브패널 · 지표 신호 판정
3. **지표조합** — 지표 레일: 30종 지표를 Lv1~Lv4 등급으로 표시/가중치 조절
4. **차트** — 가격 차트 + 예측선(1차 종합·2차 선택지표·3차 최대역치)

**지표 30종** (등급별)

| 등급 | 지표 |
|---|---|
| Lv1 핵심 | 이동평균 · MACD · RSI · 볼린저 · 거래량 |
| Lv2 주요 | 추세선 · ADX · 스토캐스틱 · 피보나치 · 일목균형표 · 피벗 · Parabolic SAR |
| Lv3 보조·전문 | VWAP · 슈퍼트렌드 · ATR · 볼륨프로파일 · 시장구조 · Keltner · Donchian · CCI · Williams %R · Aroon · MFI |
| Lv4 고급·심화 | 엘리어트 · 스마트머니 · 사이클 · 파동스캔 · ROC · Awesome · CMF |

- 지표 노드를 캔버스에 조립하면 브라우저 엔진(`forge-core.js`)이 예측 경로·합성 시그널·국면을 계산합니다.
- **지표 편집창**: 파라미터 + 도구 안내(목적·정의·해석법) + 추천값 세팅·저장 버튼.
- **웹분석**(즉시 계산) / **시뮬레이션**(지표별 작도 애니메이션 재생).
- 티커 노드로 실제 OHLC를 불러와(자체 캔들 렌더) 분석할 수 있습니다.

## 스쿱보드 (Scoop Board)

자유 캔버스에 노드·연결선·그룹으로 다이어그램을 작성합니다. 아이콘 20종, 썸네일, 정렬·자석 스냅, 직각/곡선 라우팅, PNG/SVG/JSON 내보내기, 되돌리기, 서버/로컬 저장 모드를 지원합니다.

---

## 파일 구조

```
map/
├── forge.html            ← 스쿱포지 (UI + 캔버스 작도)
├── forge-core.js         ← 스쿱포지 분석 엔진 (UMD · node 단위테스트 대상)
├── forge-core.test.js    ← 엔진 단위테스트 (node --test, 183 케이스)
├── forge-api.php         ← 스쿱포지 서버 저장 + 티커 OHLC 프록시
├── forge-guide.html      ← 엔진 작동원리 설명서(라이브 데모)
├── map.html              ← 스쿱보드 (단일 파일)
├── api.php               ← 스쿱보드 서버 저장 API
├── chart.html            ← PHASE-FOLD 차트(스쿱포지 차트의 기반)
├── docs/                 ← 설계·구현 계획 문서(specs / plans)
├── CLAUDE.md             ← 개발 컨텍스트 가이드
└── *.json · *_key.txt    ← 서버 관리 데이터(배포 시 덮어쓰지 말 것)
```

---

## 개발

- 바닐라 JS, 빌드툴·프레임워크 없음. 단일 HTML 유지(스쿱포지는 분석 엔진만 `forge-core.js`로 분리 — 유일하게 `node --test forge-core.test.js`로 단위테스트).
- 다크 테마 + 골드(`--gold`) 토큰. UI 텍스트 한국어.
- 상태는 메모리 보관, 영속화는 서버(PHP + JSON) 또는 JSON 내보내기/불러오기.
- 상세 규칙·구조·구현 메모는 [`CLAUDE.md`](CLAUDE.md), 기능별 설계는 `docs/superpowers/specs|plans/` 참조.
